# Global Meteorite Landings Analysis ☄️🗺️

A Python data science project focused on cleaning, analyzing, and visualizing global meteorite landing data. This project explores high-density impact zones across the world and analyzes human observation patterns.

## 🚀 Key Insights
* **The Map Analysis:** Our geographical visualizations reveal significant data clusters in North America and Europe, highlighting differences in historical recording and documentation rather than actual impact distribution.
* **Observation Status:** The majority of recorded meteorites were found long after their impact (**Found**), compared to a smaller percentage that were actively witnessed falling from the sky (**Fell**).
* **Missing Data Fixed:** Successfully handled over 7,000 missing coordinate cells and updated invalid weights using **Pandas** (`.dropna()` and `.fillna()`).

## 🛠️ Tech Stack & Libraries
* **Python** (Jupyter Notebook)
* **Pandas** & **NumPy** (Data Cleaning & Manipulation)
* **Matplotlib** & **Seaborn** (Advanced Data Visualization)
